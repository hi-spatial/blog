Mapbox GL JS adalah library JavaScript yang menggunakan WebGL untuk rendering peta dengan performa tinggi. Library ini memungkinkan Anda membuat peta 3D yang spektakuler.

## Persiapan

Daftar di [mapbox.com](https://mapbox.com) untuk mendapatkan access token gratis.

```html
<!DOCTYPE html>
<html>
<head>
    <script src='https://api.mapbox.com/mapbox-gl-js/v3.0.0/mapbox-gl.js'></script>
    <link href='https://api.mapbox.com/mapbox-gl-js/v3.0.0/mapbox-gl.css' rel='stylesheet' />
    <style>
        #map { width: 100%; height: 100vh; }
    </style>
</head>
<body>
    <div id="map"></div>
    <script src="app.js"></script>
</body>
</html>
```

## Inisialisasi Peta

```javascript
mapboxgl.accessToken = 'YOUR_ACCESS_TOKEN';

const map = new mapboxgl.Map({
    container: 'map',
    style: 'mapbox://styles/mapbox/dark-v11',
    center: [106.8456, -6.2088],
    zoom: 15,
    pitch: 45,
    bearing: -17.6
});
```

## Menambahkan Terrain 3D

Untuk menampilkan terrain dengan elevasi:

```javascript
map.on('load', () => {
    map.addSource('mapbox-dem', {
        'type': 'raster-dem',
        'url': 'mapbox://mapbox.mapbox-terrain-dem-v1',
        'tileSize': 512,
        'maxzoom': 14
    });
    
    map.setTerrain({ 'source': 'mapbox-dem', 'exaggeration': 1.5 });
});
```

## Building Extrusion (Bangunan 3D)

Mapbox dapat menampilkan bangunan dalam 3D:

```javascript
map.on('load', () => {
    map.addLayer({
        'id': '3d-buildings',
        'source': 'composite',
        'source-layer': 'building',
        'filter': ['==', 'extrude', 'true'],
        'type': 'fill-extrusion',
        'minzoom': 15,
        'paint': {
            'fill-extrusion-color': '#aaa',
            'fill-extrusion-height': ['get', 'height'],
            'fill-extrusion-base': ['get', 'min_height'],
            'fill-extrusion-opacity': 0.6
        }
    });
});
```

## Menambahkan Sky Layer

Untuk efek langit yang realistis:

```javascript
map.on('load', () => {
    map.addLayer({
        'id': 'sky',
        'type': 'sky',
        'paint': {
            'sky-type': 'atmosphere',
            'sky-atmosphere-sun': [0.0, 90.0],
            'sky-atmosphere-sun-intensity': 15
        }
    });
});
```

## Interaksi dengan Bangunan

```javascript
map.on('click', '3d-buildings', (e) => {
    new mapboxgl.Popup()
        .setLngLat(e.lngLat)
        .setHTML(`<strong>Tinggi:</strong> ${e.features[0].properties.height}m`)
        .addTo(map);
});

// Cursor pointer saat hover
map.on('mouseenter', '3d-buildings', () => {
    map.getCanvas().style.cursor = 'pointer';
});

map.on('mouseleave', '3d-buildings', () => {
    map.getCanvas().style.cursor = '';
});
```

## Animasi Fly To

Buat navigasi yang smooth:

```javascript
map.flyTo({
    center: [106.8272, -6.1754], // Monas
    zoom: 16,
    pitch: 60,
    bearing: 30,
    duration: 3000
});
```

## Tips Performa

1. **Gunakan `minzoom`** untuk layer yang detail
2. **Limit fitur** yang ditampilkan dengan filter
3. **Optimize source** untuk data besar
4. **Test di mobile** - WebGL bisa berat

## Kesimpulan

Mapbox GL JS membuka kemungkinan visualisasi peta 3D yang spektakuler. Dengan fitur terrain, building extrusion, dan animasi, Anda dapat membuat pengalaman peta yang immersive.
